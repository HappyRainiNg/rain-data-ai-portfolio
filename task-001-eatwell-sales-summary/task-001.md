# Task 001: Eat Well Sales Summary

## Date

2026-06-29

## Objective

This task is a beginner-level data analysis project using Python, Pandas, and Matplotlib.  
The goal is to simulate Eat Well seafood sales data, calculate revenue, create summary tables, and visualize monthly revenue.

## Tools Used

- Google Colab
- Python
- Pandas
- Matplotlib

## What I Did

- Generated simulated seafood sales data
- Saved the data as a CSV file
- Loaded and checked the dataset using Pandas
- Added month and revenue columns
- Sorted orders by revenue
- Created summary tables by month, category, product, and customer type
- Created a monthly revenue bar chart

## Key Findings

- June 2026 had the highest revenue.
- Lobster was the highest revenue product.
- Restaurant customers contributed the highest revenue.
- Shellfish was the strongest category across the simulated dataset.
