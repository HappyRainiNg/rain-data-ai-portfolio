# Eat Well Sales Summary

Date: 2026-06-29

## Project Goal

This project uses Python pandas to create a simple sales analysis prototype for Eat Well seafood business.

## Dataset

The dataset is a simulated seafood sales dataset with at least 50 orders.

Columns:
- order_id
- order_date
- product
- category
- quantity
- unit_price
- customer_type
- district
- revenue
- month

## Analysis

This notebook produces:

1. Monthly revenue summary
2. Monthly category revenue
3. Product ranking
4. Customer type revenue contribution
5. Monthly revenue bar chart

- Python
- pandas
- groupby
- data cleaning

- business insight
